#### 2.1 数据的输入

**语法：**`cin>>变量;`

```C++
int a=0;
cin>>a;

string str="hello";
cin>>str;
```



#### 2.2 运算符

| 运算符      | 术语       | 示例        | 结果      |
| ----------- | ---------- | ----------- | --------- |
| +、-、\*、/ | 加减乘除   |             |           |
| %           | 取模(取余) | 10%3        | 1         |
| ++          | 前置递增   | a=2; b=++a; | a=3; b=3; |
| ++          | 后置递增   | a=2; b=a++; | a=3; b=2; |
| --          | 前置递减   | a=2; b=--a; | a=1; b=1; |
| --          | 后置递减   | a=2; b=a--; | a=1; b=2; |

##### 赋值运算符

`a=a+2`   等效于  `a+=2`

`a=a*2`   等效于  `a*=2`

`a=a/2`   等效于  `a/=2`

`a=a%2`   等效于  `a%=2`

##### 比较运算符

`==、!=、<、>、<=、>=`

##### 逻辑运算符

| 运算符 | 术语 | 示例   |
| ------ | ---- | ------ |
| !      | 非   | !a     |
| &&     | 与   | a&&b   |
| \|\|   | 或   | a\|\|b |
